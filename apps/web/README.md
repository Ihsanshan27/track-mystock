# Frontend App

Frontend `Jurnal Saham` berbasis React + Vite.

Folder utama:

- `src` untuk source code aplikasi
- `public` untuk aset statis
- `index.html` untuk entry HTML Vite
- `vite.config.ts` untuk konfigurasi frontend

Perintah harian tetap dijalankan dari root project:

```bash
npm run dev
npm run build
npm run preview
```
